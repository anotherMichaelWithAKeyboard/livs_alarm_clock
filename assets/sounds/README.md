# Alarm Sounds

Place your custom alarm sound files in this directory.

## Supported Formats
- MP3 (`.mp3`)
- WAV (`.wav`)
- OGG (`.ogg`)

## Adding Sounds
1. Copy your audio files to this directory
2. They will automatically appear in the alarm sound selection menu

## Example Files
You can download free alarm sounds from:
- [Freesound.org](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
- [SoundBible](http://soundbible.com/)

Make sure to check the license for any sounds you download.
